  SBDI is excited to announce that SUBSIM, the Swedish platform for subsea image analysis, is now available through the European Digital Twin Ocean infrastructure EDITO.
---

A new tutorial demonstrates how computer vision methods can be used for marine biodiversity analysis using underwater footage collected on the Swedish west coast. Users are guided through data preparation, model training, testing, and evaluation directly within the EDITO environment.

The workflow is designed as an introductory demonstration for SBDI users learning computer vision methods in EDITO, while the full SUBSIM workflows available through the SUBSIM GitHub pages provide more advanced functionality and analysis options.
